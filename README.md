### Features
- Log planned vs actual tasks every day (2 clicks)
- AI automatically discovers:
  - Your most productive hours
  - What distracts you the most
  - Why you abandon certain tasks
- Generates a personalized weekly plan with 85–90% predicted success rate
- Beautiful interactive dashboard with Plotly
- Machine Learning model that gets smarter the more you use it
